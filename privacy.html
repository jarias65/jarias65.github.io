Política de Privacidad

Última actualización: junio 2025

Esta guía de estudio es un recurso educativo de uso libre. No recopilamos, almacenamos ni compartimos ningún dato personal. Todo funciona directamente en tu navegador.

1. Información General

La PMI-ACP Guía Completa de Estudio ("la Guía") es una aplicación web estática de carácter educativo, diseñada para ayudar a profesionales a prepararse para la certificación PMI-ACP (Agile Certified Practitioner) del Project Management Institute.

Esta Política de Privacidad describe cómo se gestiona la información cuando utilizas la Guía, publicada a través de GitHub Pages.

2. Datos que NO Recopilamos

La Guía no recopila ningún tipo de dato personal. Específicamente:

  • No solicitamos nombre, correo electrónico ni ningún dato de identificación
  • No usamos cookies propias ni rastreadores de terceros
  • No integramos Google Analytics, Facebook Pixel ni ninguna herramienta de analítica
  • No creamos cuentas de usuario ni perfiles
  • No enviamos datos a ningún servidor externo
  • No almacenamos información en bases de datos remotas

3. Almacenamiento Local

El progreso de estudio (módulos vistos, temas marcados) se gestiona únicamente en la memoria del navegador durante la sesión activa. Al cerrar la pestaña o el navegador, dicha información no se conserva.

La Guía no utiliza localStorage, sessionStorage ni IndexedDB para retener información entre sesiones.

4. Fuentes Externas

La Guía carga recursos tipográficos desde Google Fonts para mejorar la experiencia visual. Esta conexión puede generar una solicitud de red a los servidores de Google, lo cual está sujeto a la Política de Privacidad de Google.

Si deseas evitar esta conexión, puedes usar la Guía descargando el archivo HTML y abriéndolo localmente sin conexión a internet, o eliminando la línea de importación de Google Fonts del código fuente.

5. GitHub Pages

La Guía está alojada en GitHub Pages, servicio de GitHub, Inc. GitHub puede registrar datos de acceso básicos como dirección IP y agente de usuario con fines operativos y de seguridad, según su propia política de privacidad.

Para más información consulta la Política de Privacidad de GitHub.

6. Derechos del Usuario

Dado que no recopilamos ningún dato personal, no existe información que rectificar, eliminar ni exportar. No obstante, si tienes alguna consulta relacionada con la privacidad de esta Guía, puedes contactarnos a través del repositorio en GitHub.

7. Propiedad Intelectual

El contenido educativo de esta Guía es de elaboración propia con fines de estudio. Las referencias a marcos de trabajo como Scrum, Kanban, SAFe, XP o Lean, y a la certificación PMI-ACP, son de sus respectivos titulares. Esta Guía no está afiliada ni patrocinada por el Project Management Institute (PMI) ni por ninguna otra organización certificadora.

8. Uso Aceptable

Esta Guía se proporciona con fines exclusivamente educativos. Queda prohibido:

  • Comercializar o redistribuir el contenido con fines de lucro sin autorización
  • Presentar el contenido como material oficial del PMI
  • Modificar el contenido de forma que induzca a error sobre los temas de certificación

9. Cambios a esta Política

Cualquier actualización a esta Política de Privacidad se reflejará en la fecha indicada al inicio de este documento. Al ser un recurso estático sin registro de usuarios, no existe mecanismo de notificación activa de cambios.

10. Contacto

¿Tienes alguna pregunta?

Puedes abrir un issue directamente en el repositorio de GitHub donde está publicada esta Guía. Responderemos a la brevedad posible.
