# PMI-ACP — Guía Completa de Estudio

Guía interactiva para preparar la certificación **Agile Certified Practitioner (PMI-ACP)**.

## 🌐 Ver en línea

Una vez activado GitHub Pages, la guía estará disponible en:

```
https://<tu-usuario>.github.io/<nombre-del-repo>/
```

## 📚 Contenido

- 11 módulos de estudio (Manifiesto Ágil, Scrum, Kanban, XP, Lean, Estimación, Backlog, Métricas, Equipos, Escalado, Riesgos)
- Flashcards interactivas
- Quiz tipo examen con 10 preguntas
- Plan de estudio de 8 semanas
- Tracker de progreso por módulo

## 🚀 Cómo activar GitHub Pages

1. Sube los archivos a un repositorio en GitHub
2. Ve a **Settings → Pages**
3. En **Branch** selecciona `main` y carpeta `/ (root)`
4. Haz clic en **Save**
5. En unos minutos tu guía estará online
