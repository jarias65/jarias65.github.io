## 📚 Contenido

- 11 módulos de estudio (Manifiesto Ágil, Scrum, Kanban, XP, Lean, Estimación, Backlog, Métricas, Equipos, Escalado, Riesgos)

- Quiz tipo examen con 10 preguntas
- Plan de estudio de 8 semanas
- Tracker de progreso por módulo
